# ふるれこ(FuruReco)
ふるさと納税をした自治体を記録するアプリ。1747自治体全制覇を目指せ！

## プレビュー
### 自治体一覧検索機能＆チェック機能
寄付した自治体にチェックをつけよう
画面上部のサーチバーから自治体名、都道府県、自治体コードで検索できるぞ！
